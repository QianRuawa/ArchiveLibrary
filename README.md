# ArchiveLibrary（档案库）
**《杀戮尖塔2》（Slay the Spire 2）工具库模组，基于 RitsuLib 框架优化制作**
[RitsuLib](https://github.com/BAKAOLC/STS2-RitsuLib)

**详细教程**
[ArchiveLibrary-优化指南](https://github.com/QianRuawa/ArchiveLibrary-Optimization-Guide)

## 内容

### 基础功能
- 日志工具 — 统一的 Mod 日志系统
- 角色 3D 场景配置 — 简化角色场景资源加载
- 多语言翻译辅助 — 本地化文件读取与扩展

### 角色视觉与动画
- 额外动画播放 — 角色动画注册与皮肤系统
- 角色音效播放 — 角色事件音效的注册与播放
- 战斗背景切换 — 动态更换战斗场景背景
- 标题动画生成 — 自定义标题动画效果
- 角色面向与房间缩放 — 获取角色朝向和场景缩放

### 音乐与音频
- Boss BGM 播放 — Boss 战专属背景音乐
- 层级音乐播放辅助 — 自定义地图层级的音乐管理

### 战斗系统
- Boss 视觉材质修改 — Boss 场景材质效果
- 固定攻击意图伤害 — 怪物固定伤害意图
- 攻击意图标题修改 — 自定义意图描述与标题
- 敌人注册控制 — 敌人注册开关控制

### 生物与伤害
- 生物位移能力 — 生物在场景中的位移
- 单正负 Power 工具 — 双模式 Power（增益/减益）
- 伤害类自动运算 — 自动化的伤害计算链
- 贯通格挡设置 — 穿透格挡的调用接口
- 临时伤害提升 — 卡牌临时伤害加成

### 卡牌与 Power
- 卡牌升级检测 — 监听卡牌升级事件
- SmartHoverTip — 双形态 Power 智能提示框
- Power 视觉追踪 — Power 施加/移除的视觉反馈

### 遗物与图鉴
- RelicHelper — 判断遗物存在与触发
- 先古遗物图鉴分类 — 自定义遗物图鉴注册

### 界面
- 额外血量显示优化 — 额外的血量展示
自定义层级
- 自定义新层级 — 支持模组创建自定义 Act
- 层级房间生成 — 自定义房间布局与生成
