# ArchiveLibrary（档案库）
**《杀戮尖塔2》（Slay the Spire 2）工具库模组，基于 RitsuLib 框架优化制作**
[RitsuLib](https://github.com/BAKAOLC/STS2-RitsuLib)

**详细教程**
[ArchiveLibrary-优化指南](https://github.com/QianRuawa/ArchiveLibrary-Optimization-Guide)

**内容**
- 可继承修改adverse/benefit/crowdControl/particular
- 自定义层级地图优化制作
*固定事件*/*替换房间图标*/*房间节点下生成特效*/*自动注册到怪物图鉴*
- 优化播放音乐[不熟悉或者不想使用Fmod制作]内制播放
- 3D玩家场景播放动画
*额外动画播放*/*角色创建自动获取Anim节点*
- 额外配置翻译
*提供其他自动化翻译*
